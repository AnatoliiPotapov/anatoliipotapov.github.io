# potapov.ai

Personal site of **Anatoly Potapov** — Head of LLM Project, leading the GEN-T family of open-source Russian-language LLMs (T-Lite, T-Pro, T-Pro 2.0, T-Pro 2.1).

Live at [potapov.ai](https://potapov.ai).

## Stack

Static HTML — single page, no build step, no JavaScript. Deployed via GitHub Pages.

## Local preview

Open `index.html` directly in a browser, or serve the directory:

```sh
python3 -m http.server 8000
```

## Deploy

This is a GitHub user-site repo: any push to `main` is deployed automatically
by GitHub Pages to [anatoliipotapov.github.io](https://anatoliipotapov.github.io)
and to [potapov.ai](https://potapov.ai) once the custom domain is bound.
